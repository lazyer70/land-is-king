遊戲規則:
	1.先將land_king.html處理host端的ip跟server.py的37行localhost設成自己的ip(從CMD下ipconfig中的IPV4即可)
	2.執行server.py
	3.選擇你的符號
	4.輸入你是玩家1還是玩家2
	5.其中1方按下start即開始
	6.每1格都有屬於自己的防禦值，按1下防禦值+1，對方按1下-1，預設是0，格子防禦值是1時是自己的符號，0是預設符號，負數時是對方的符號
	7.遊戲結束後格子多的玩家勝利
注意事項:
	1.由於是socket連線因此要在同樣網域下
	2.遊戲結束後需要重新整理才能再次遊玩
	3.玩家輸入自己是幾號玩家時，時要注意內容不能有空白
	4.偶爾會發生一些不可預期的錯誤，可以按F12去確認是否連線，如果確認有連線，可以重新整理再試試看，若沒連線可能是IP輸錯或防火牆問題
程式架構:
	1.下方的html是建立格子跟輸入表格等要素
	2.doInit()是建立連線跟處理接收訊息像是
	3.在下方html的start按鈕按下去會start() function，function則是傳給server時間的訊息跟修改時間
	4.時間為0時會做個總結，並判別誰獲勝
	5.Contest() function是在偵測到開始後執行，其實玩家1的defense值會是往上+，玩家2是往下-，由此分辨該用誰的符號 setCell()
	以上為簡單的介紹
